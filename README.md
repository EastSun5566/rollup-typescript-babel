# Rollup TypeScript Babel library Starter

> This starter is base on <https://github.com/a-tarasyuk/rollup-typescript-babel>

## Building the repo

```sh
npm run build
```

## Building only types

```sh
npm run build:types
```

## Type-Checking the repo

```sh
npm run type-check
```

And to run in --watch mode:

```sh
npm run type-check:watch
```
